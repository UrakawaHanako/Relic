Requirement

Node.js required :
1. Download and install Node.js from nodejs.org
2. Run "npm npm install -g http-server"
3. then http-server

Common Errors

ConnectionResetError: [WinError 10054] An existing connection was forcibly closed by the remote host
Artinya browser yang nutup koneksi secara paksa, bukan Python yang “crash duluan”.

Pada http.server (Python built-in), ini sangat umum terjadi ketika:
Browser:
membatalkan request (tab ditutup / reload)
seek video (lompat timeline)
autoplay → ganti src cepat
Server:
mengirim file besar (mp4) tapi tidak mendukung range request dengan baik
